#GVEL

### Installation
```shell script
go build .
```

### Usage
#### Help
```shell script
./gvel --help
./gvel init --help
./gvel account --help
```

#### Initialization
```shell script
./gvel init
```
#### Create account
```shell script
./gvel account create

# set as default account
./gvel account create --default
# or
./gvel account create -d
```
#### List account
```shell script
./gvel account list
```
